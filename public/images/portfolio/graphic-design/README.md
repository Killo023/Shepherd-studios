## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 1200x900px or larger
- **Aspect Ratio**: 4:3 (will be cropped to fit)
- **File Size**: Optimize images to keep file sizes reasonable (< 2MB per image)

## Notes

- Images will automatically fall back to a placeholder if they're not found
- The component supports both JPG and PNG formats
- Images are optimized by Next.js automatically
