# Service Images

Place your service images in this directory.

## Required Images

Please add the following images with these exact filenames:

1. **motion-graphics.jpg** - Image representing motion graphics/animation services
2. **videography.jpg** - Image representing videography and media production services
3. **graphic-design.jpg** - Image representing graphic design services

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 600x400px or larger
- **Aspect Ratio**: 3:2 (will be displayed at 4:3 ratio)
- **File Size**: Optimize images to keep file sizes reasonable (< 500KB per image)

## Notes

- Images will automatically fall back to a placeholder if they're not found
- The component supports both JPG and PNG formats
- Images are optimized by Next.js automatically
